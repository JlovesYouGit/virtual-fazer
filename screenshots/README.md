# RedBird App Screenshots

This folder contains automated screenshots of the RedBird social media application.

## Screenshots

- `login-page.jpg` - Login page with Guest Mode
- `feed-page.jpg` - Main feed with posts
- `reels-page.jpg` - Reels/TikTok-style video feed
- `notifications-page.jpg` - Notifications center
- `profile-page.jpg` - User profile page

## How to Generate

Run the screenshot capture script:

```bash
node screenshot.js
```

Requirements:
- Frontend server running on http://localhost:5178
- Playwright installed: `npm install -g playwright`
