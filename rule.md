# Instagram-like Neural Network Project Rules

## Project Overview
Building a social media platform with intelligent user categorization and auto-following capabilities using a neural interface backend.

## Core Architecture
- **Multi-language backend**: Python Django (main), Go (high-performance), Ruby (services), Node.js (real-time)
- **Neural Interface**: AI-powered user categorization based on behavior patterns
- **Fast Memory Parsing**: Cobol integration for database operations
- **Auto-follow System**: Intelligent user matching and connection system

## Project Goals

### Primary Objectives
1. **User Management System**
   - Registration and authentication
   - Profile management
   - User categorization based on behavior

2. **Neural Categorization Engine**
   - Real-time behavior analysis
   - Pattern recognition and learning
   - Dynamic category assignment

3. **Connection System**
   - Smart user matching
   - Auto-follow similar users
   - Network graph analysis

4. **Communication Features**
   - Chat system
   - Inbox management
   - Interaction tracking

### Technical Requirements
- Scalable microservices architecture
- Real-time data processing
- Efficient caching strategies
- Cross-service API communication

## Technology Stack

### Backend Services
- **Python/Django**: Main application logic and API
- **Go**: High-performance services and data processing
- **Ruby**: Background jobs and utility services
- **Node.js**: Real-time communication and websockets
- **Cobol**: Fast memory parsing and database operations

### Database
- **PostgreSQL**: Primary user data
- **Cassandra**: Activity feeds and time-series data
- **Redis**: Caching and session management
- **Elasticsearch**: Search and analytics

### Infrastructure
- **Docker**: Containerization
- **Kubernetes**: Orchestration
- **Nginx**: Load balancing
- **RabbitMQ**: Message queuing
